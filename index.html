Quickstart guide

Commands:

  • create w,h: creates a new grid of width (w) and height (h)
  • clear: clears current grid
  • run: begins running CGOL simulation
  • stop: stops currently running simulation
  • help: opens the panel you're seeing right now

Errors:

  • command not found: pretty self explanatory
  • check command name / params: you probably didn't format your command correctly
