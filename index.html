Quickstart guide

Commands:

  • create w,h: creates a new grid of width (w) and height (h)
  • clear: clears current grid
  • run: begins running CGOL simulation
  • stop: stops currently running simulation
  • help: opens the panel you're seeing right now

Keybinds

  • [s]: perform one step of the simulation
  • [c]: clears current grid

Errors:

  • command not found: pretty self explanatory
  • check command name / params: you probably didn't format your command correctly

Grid sizes

  • You can make uneven grid sizes however they dont look great so it's not recommended.
  • Making a grid that's too big can crash the website so do with that what you will ¯\_(ツ)_/¯

Placing tiles

  • Click anywhere on the canvas to place a tile, and click again to remove it
